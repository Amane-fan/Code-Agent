# 产品说明

## 目标

构建一个终端优先的 AI 编程 Agent。它能够检查本地代码仓库、规划代码修改、生成可审查补丁，
并在需要时应用补丁、运行测试，最后总结执行结果。

## MVP 用户故事

- 作为开发者，我可以用自然语言询问仓库问题，并得到结合仓库上下文的计划。
- 作为开发者，我可以通过 LLM Provider 为指定修改生成 unified diff。
- 作为开发者，我可以要求补丁先通过校验，再真正修改工作区。
- 作为开发者，我可以在补丁应用后运行自动检测到的测试命令。

## 成功标准

- CLI 可以通过 `uv sync` 安装依赖后稳定运行。
- Agent 工作流由 LangGraph `StateGraph` 编排，节点和条件边清晰可测。
- `.env` 等敏感文件不会进入模型上下文，也不能被文件工具直接读取。
- 生成的补丁只有在 `git apply --check` 通过后才会被应用。
- 每次运行都可以保存为本地 JSON 会话日志，便于审计和复盘。
